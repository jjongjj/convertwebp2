📖 ConvertWemp 사용 방법

🎯 가장 쉬운 방법: 더블클릭으로 실행
==========================================

1. "start-convertwemp.command" 파일을 더블클릭하세요
   (처음 실행시 필요한 프로그램들을 자동으로 설치합니다)

2. ConvertWemp 창이 열리면:
   - GIF 파일을 드래그해서 창에 떨어뜨리거나
   - "파일 선택" 버튼을 클릭해서 GIF 파일들을 선택하세요

3. 설정 조정 (선택사항):
   - 품질: 높을수록 화질 좋음, 파일 크기 큼
   - 압축 노력도: 높을수록 더 작은 파일 크기
   - 프리셋 버튼: 자동최적화, 고품질, 고압축, 기본설정

4. "▶️ 변환 시작" 버튼을 클릭하세요

5. 변환 완료 후 "📁 결과 폴더 열기"를 클릭해서 
   변환된 WebP 파일들을 확인하세요

💡 팁:
- WebP 파일은 GIF보다 60% 이상 작은 크기
- 화질은 거의 동일하게 유지
- 여러 파일을 한번에 변환 가능

❓ 문제가 있다면:
- Node.js가 설치되어 있는지 확인 (https://nodejs.org)
- 터미널을 열고 "npm run gui" 입력해서 실행
- 또는 개발자에게 문의
